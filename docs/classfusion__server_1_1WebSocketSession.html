This method allows the current writing to complete (if any) and then closes the connection. After its called no writing should be performed.

◆ Write()

void fusion_server::WebSocketSession::Write ( Package  package)
noexcept

This method delegates the write operation to the client. Since Boost::Beast allows only one writing at a time, the packages is always queued and if no writing is taking place the asynchronous write operation is called.

Parameters
[in]packageThe package to be send to the client. The shared_ptr is used to ensure that the package will be freed only if it has been sent to all the specific clients.
Note
This method is thread-safe.
